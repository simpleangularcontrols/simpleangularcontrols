positionElements
positionElements(hostElement: HTMLElement, targetElement: HTMLElement, placement, appendToBody?: boolean, baseClass?: string)

Accept the placement array and applies the appropriate placement dependent on the viewport. Returns the applied placement. In case of auto placement, placements are selected in order 'top', 'bottom', 'left', 'right', 'top-left', 'top-right', 'bottom-left', 'bottom-right', 'left-top', 'left-bottom', 'right-top', 'right-bottom'.
